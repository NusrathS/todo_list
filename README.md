# Glassmorphism To-Do App

![Uploading image.png…]()

## Features

### Adding Tasks
- You can type a task and click the `+` button or press `Enter` to add it to the list

### Completing Tasks
- Each task has a circle checkbox on the left
- When you check it, the task gets a strikethrough and turns black showing it's done
- The edit button gets disabled so you can't edit a completed task

### Editing Tasks
- Each task has a yellow pen button
- Clicking it puts the task text back into the input box so you can retype and re-add it

### Deleting Tasks
- Each task has a pink trash button
- Clicking it removes that task from the list

### Empty State
- When there are no tasks, an empty image shows up
- When tasks are added, the image hides automatically

### Progress Tracker
- At the top there is a pink progress bar that fills up as you complete tasks
- Next to it a circle shows how many tasks are done out of total like `2 / 5`

### Celebration
- When you complete all tasks, a confetti animation fires on the screen

### Saves Your Tasks
- All your tasks are saved in the browser's local storage
- So even if you close and reopen the browser, your tasks are still there
